# Repository for IBM Data Analyst Certificate: Capstone Project Deliverables

## Goal
This repository documents the achievements made in Nicholas Pater's submission for the IBM Data Analyst Capstone Project, 1 course out of 11 courses required for the IBM Data Analyst Professional Certificate.

## Project Description
### Scenario
A global IT and business consulting firm wishes to analyze data to identify emerging and future skill requirements.
### Data Sources
- Results of the latest Stack Overflow Developer Survey
- Web scraping of job postings and training portals for data on popular programming languages
- Third party api for collecting job postings data
